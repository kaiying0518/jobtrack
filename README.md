# JobTrack

JobTrack は、就職活動の応募状況をローカルで整理・記録するための Web アプリです。  
応募一覧、選考状況、採用ページ情報、活動記録、簡単な振り返りを、少ない入力でまとめて管理できるようにしています。

## 概要

就職活動では、応募先が増えるほど「今どこがどういう状況か」「次に何をするべきか」が見えにくくなりがちです。  
JobTrack は、そうした情報を一か所にまとめて、流れを把握しやすくすることを目的に作成しました。

クラウド前提ではなく、ローカル環境で動かすことを想定しており、個人利用でも扱いやすい構成を意識しています。

## 主な機能

- 応募情報の登録・編集・削除
- 選考ステータスの更新
- 採用ページ・ログイン情報の管理
- 活動記録の自動保存
- 振り返りページでの簡単な集計表示
- CSV エクスポート
- H2 Database を使ったローカル保存

## 画面イメージ

- 応募一覧
- 応募詳細
- 活動記録
- 振り返り
- 設定

※ スクリーンショットは今後追加予定です。

## 使用技術

- Java 21
- Spring Boot
- Thymeleaf
- Spring Data JPA
- H2 Database
- Gradle

## 設計の考え方

このアプリでは、できるだけ入力負担を増やしすぎず、必要な情報だけを整理して見返せることを重視しています。  
単に記録するだけでなく、今の状況を把握しやすくし、次の行動を考えやすくすることを意識しています。

## データ保存について

- データは H2 Database を使ってローカルに保存されます
- 現在はローカル利用を前提としており、ユーザー認証機能は実装していません
- 応募情報や活動記録を、個人で無理なく整理できる形を目指しています
