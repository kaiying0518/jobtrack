# JobTrack

JobTrack は、就職活動の応募状況・選考進捗・振り返りを、ローカル環境で管理するための Windows 向けアプリです。

応募先が増えるほど見えにくくなる「現在の状況」と「次にやるべきこと」を整理し、少ない入力で就職活動全体を把握しやすくすることを目的としています。

## 主な機能

### 応募管理

- 応募情報の登録・編集・削除
- 企業名、職種、応募経路、求人URL、メモの管理
- 選考ステータスの管理
- 次回予定日時の記録

### ダッシュボード

- 総応募数の表示
- 選考ステータス別の状況確認
- 最近更新した応募の表示
- AI Summary の表示

### 振り返り・AI機能

- 応募状況の整理
- AI Summary による簡易的な振り返り支援
- AI Chat による応募状況の相談・確認
- チャット履歴の保存・管理

### 設定・出力

- デフォルト入力値の設定
- AIプロバイダー、APIキー、モデル等の設定
- CSV エクスポート
- ログ出力

---

## 起動方法

1. ダウンロードした ZIP ファイルを解凍します
2. `JobTrack.exe` をダブルクリックします
3. 起動後、自動的にブラウザが開きます

自動で開かない場合は、以下のURLを手動で開いてください。
http://localhost:8080
